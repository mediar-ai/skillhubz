# skillhubz

Computer Use Skills Marketplace — browse, share, and install automation skills.

## Structure

```
skillhubz/
├── apps/
│   └── web/              # skillhu.bz website (React + Vite)
├── packages/
│   ├── cli/              # skillhu CLI (npm package)
│   └── skills/           # Skill definitions registry
```

## Quick Start

```bash
# Install dependencies
npm install

# Run website locally
npm run dev

# Build website
npm run build
```

## CLI Usage

```bash
npx skillhu install gmail-reply
```
